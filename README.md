## 📚 Audible Insights — Smart Book Recommender System

**Audible Insights** is an interactive, data-driven **book recommendation system** built using **Machine Learning and Streamlit**. It helps users discover books tailored to their preferences using a hybrid recommendation approach that combines **content-based filtering** and **ratings analysis**.

Designed with a modern UI and analytics dashboard, the application not only suggests books but also provides deep insights into trends such as ratings, genres, authors, and pricing.

---

## 🚀 Key Features

* 🔍 **Smart Book Recommendations**
  Uses TF-IDF and cosine similarity to recommend books similar to user-selected titles.

* 🧠 **Hybrid Recommendation Model**
  Combines:

  * Content similarity (TF-IDF)
  * User ratings for better accuracy

* 🎯 **Advanced Filtering System**

  * Filter by genres
  * Search by author
  * Search by book title

* 🖼 **Dynamic Book Cover Fetching**
  Fetches real-time book thumbnails using Google Books API.

* 📊 **Interactive Analytics Dashboard**

  * Rating distribution
  * Top authors
  * Genre popularity
  * Reviews vs Rating
  * Price vs Rating

* 🎨 **Premium UI Design**

  * Glassmorphism cards
  * Smooth hover effects
  * Clean dark theme

---

## 🧠 How It Works

The recommendation system uses:

* **TF-IDF Vectorization** to convert book metadata into numerical features
* **Cosine Similarity** to find similar books
* A **hybrid scoring system**:

  * 70% similarity score
  * 30% normalized rating score

This ensures recommendations are both **relevant and high-quality**.
